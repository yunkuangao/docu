## 描述
三阶魔方
[nonebot-plugin-cube](https://github.com/initialencounter/nonebot-plugin-cube)重写版

## 使用方法
- 操作魔法
  - cb +【f,b,u,d,l,r】不区分大小写，在字母前加入【非方向字符】代表顺时针旋转
- 新建魔法
  - cb
- 自定义魔法
  - cb.def 魔方数据
- 撤销操作
  - cb.back
- 魔方排行榜
  - cb.rank
- py交易 修改群友权限
  - cb.bind+一次性key 
  - 在配置项设置一次性key，交易对象发给机器人，
