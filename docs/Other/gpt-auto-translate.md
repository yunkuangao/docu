## 1. 安装python依赖

打开server目录，按住shift+鼠标右键，打开终端或powershell。

![](../assets/gpt-auto-translate/1.png)

输入命令 `pip install -r requirements.txt` 回车。

![](../assets/gpt-auto-translate/2.png)

这样显示则代表安装成功。

![](../assets/gpt-auto-translate/3.png)
